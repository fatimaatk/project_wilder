Que faisons-nous à la Wild?

Langages travaillés :
Pendant votre formation, vous approfondirez vos connaissances en HTML et CSS, mais vous apprendrez aussi les bases de JavaScript ainsi que PHP !
Après vos deux premières semaines, on vous demandera de choisir le langage avec lequel vous souhaitez perfectionner au cours de la formation, qu'allez vous choisir ?
Workshops :
Au cours de la formation, différents ateliers seront mis en place, ces derniers peuvent se dérouler seul ou en groupe.
Ces ateliers vous permettront d'appliquer certaines notions vu lors des cours, mais aussi de vous familiariser avec certaines erreurs, afin de les éviter, ou de savoir les réparer.
Notez qu'il est possible de les refaire à n'importe quel moment !
Quêtes :
Les quêtes représentent une grosse partie du cursus, vous en aurez des nouvelles toutes les semaines.
Ces dernières, vous permettront de voir certaines notions, et de les mettre en pratique (du moins essayer) avant le cours les concernants. Certaines de ces quêtes auront une solution à rendre, vous serez alors corrigés par ves camarades.
Il est donc TRES important de les faire sérieusement !
Projets :
Entre vos cours, workshops et vos quêtes, on vous demandera de réaliser des projets.
Ces derniers se réaliseront en groupes, afin de vous permettre d'apprendre les un des autres, et surtout de vous familiariser avec les outils de développeur.
Vous devrez passer par différentes étapes : La création de maquettes (Figma peut être utile pour cela), la réalisation des parties communes à vos pages ( NavBar, banniere s'il y a, footer, ... ), et enfin le contenue de vos pages.
Il sera donc très important de bien s'organiser et discuter avec ses collègues.
Attentes :
Et oui, la Wild Code School, ce n'est pas une promenade, on attend beaucoup de vous !
Voici une petite idée de ce qui vous attends si vous choisissez de continuer le JavaScript :
Semaine 1 - 3 HTML/CSS/JS Basics - Projet 1
Semaine 4 : Introduction POO (Programmation Orientée Objets)
Semaine 5 Checkpoint 1 HTML / CSS / JS
Semaine 6 : Start React + Projet 2 sur React + call API + Introduction Backend Node.JS Express
Semaine 10 : Rendu Projet 2
Semaine 11 : Checkpoint 2 React + API // Hackathon 1
Semaine 12 : Introducion Base de données + Node.Js Express MVC
Semaine 13 : Start Projet Client Fullstack
Semaine 16 : Checkpoint 3 (Node.JS)
Semaine 17 : Hackathon 2
Semaine 20 : Final Checkpoint (React + Node) + Livraison Projet Client
